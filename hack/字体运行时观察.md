## 3. 运行时字体上下文

在 `0x020869C4` 处读取 `DAT_020B73E0`：

```text
020B73E0:
06880000 06881C00 02282EA0 00000004
00000000 00000000 00000000 00000000
0688E6C0 000000E0 0688EA40 0000032B
00000000 06206000 06200000 00000000
```

字段解释：

- `+0x00 = 0x06880000`：`font_1x1.chr` VRAM 基址。
- `+0x04 = 0x06881C00`：`font_1x2.chr` VRAM 基址。
- `+0x08 = 0x02282EA0`：`font.plt` RAM/堆指针。
- `+0x0C = 0x00000004`：palette 相关计数。
- `+0x20 = 0x0688E6C0`：`font_1x1.tbl` VRAM 指针。
- `+0x24 = 0x000000E0`：`font_1x1.tbl` 项计数。
- `+0x28 = 0x0688EA40`：`font_1x2.tbl` VRAM 指针。
- `+0x2C = 0x0000032B`：`font_1x2.tbl` 项计数。

## 5. 查表函数静态确认

`0208671C` 的关键逻辑：

- `r0`：字符编码。
- `r1`：字体模式/尺寸索引。
- 从 `020B7400 + mode * 8` 取 `.tbl` 指针。
- 从 `020B7404 + mode * 8` 取 `.tbl` 项计数。
- 调用 `020866BC` 做 4 字节表项二分查找。
- 查到后从 `020B73E0 + mode * 4` 取 `.chr` 基址。
- 从 `020AC480 + mode * 4` 取单位大小：
  - mode 0：`0x20`
  - mode 1：`0x40`
- 返回：`chr_base + unit_size * glyph_index`。

## 7. 已确认的运行时绘制链路

实际链路：

```text
0208913C  文本绘制入口
  -> 0208916C  bl 0208671C
  -> 0208671C  字符编码 + mode 查表
  -> 02089170  查表返回，R0 为 glyph 源地址
  -> 02089190  调用 020087BC 复制 glyph
```

第一轮命中样本：

```text
0208913C:
R0=0x0229CF60
R1=0x00008140
LR=0x02088620

0208671C:
R0=0x00008140
R1=0x00000001
LR=0x02089170

02089170:
R0=0x06881C00
R1=0x00000040

02089190:
R0=0x06881C00
R1=0x02292A40
R2=0x00000040
```

解释：

- `0208913C` 的 `R1` 是当前字符编码。
- `0208671C` 的 `R0` 是当前字符编码，`R1` 是字体 mode。
- 当前标题/菜单路径使用 `mode = 1`，即 1x2 字体。
- `0208671C` 返回的 `R0` 是 glyph 源地址。
- `02089190` 调用复制函数时：
  - `R0` 是 glyph 源地址；
  - `R1` 是目标缓存地址；
  - `R2` 是复制大小；
  - 本轮为 `0x40` 字节。

公式已动态确认：

```text
glyph_source = chr1x2_base + glyph_index * 0x40
```

其中 `chr1x2_base = 0x06881C00`。

## 9. 对 patch 点的判断

当前最小风险 patch 点有两个：

- `0208671C`：改变字符编码到 glyph 源地址的返回值。
- `02089190`：在复制 glyph 前，按当前字符准备/替换 `R0` 源地址。

`02089190` 更接近实际复制点，输入输出清楚，适合作为第一版最小验证点。后续正式动态缓存仍可能需要回到 `0208671C`，因为这里能统一控制字符到 glyph 的映射。

下一步需要：

- 第一版 hook 已在 `rom/test_vram_font_hook_probe_arm9.nds` 验证：`02089190` 跳到 `0207411C`，再进入 `020087BC` 时 `R0` 已替换为 `0x0207413C`。
- 后续进入动态缓存设计：按当前字符码选择/准备 glyph，而不是只按固定源地址替换。
